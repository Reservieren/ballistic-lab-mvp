
# Ballistic Lab — MVP

Protótipo mínimo para calcular distância e azimute entre dois pontos em um mapa.
Escala usada: **783 px × 768 px = 4000 m × 4000 m**.

## Como rodar
1. Clone o repositório:
   ```bash
   git clone <seu-repo-url>
   cd ballistic-lab-mvp
2. Coloque o arquivo de mapa em assets/map.png (resolução original 783×768 px).

3. Abra index.html no navegador (ou use Live Server no VS Code).

Tecnologias

HTML5 + Canvas

CSS puro

JS (ES6)
